
private key is never written
it is generated from mneumonic on the cold storage computer (csc)
that computer's hard drive is encrypted

public key for main account goes on node

Withdraw Process:
  Wallet client on phone generates output address and corresponding QR code (copay does this)
  node scans QR code
  node looks up utxos that belong to addresses on the main account
  node creates transaction with utxos and output address
  node saves transaction to flash drive
  user inserts flash drive into csc
  csc adds change address to the transaction and keeps track of which ones it has used
  csc asks user to verify address
  csc signs transaction and saves it to flash drive
  user shuts down csc
  user inserts flash drive back into node
  node publishes transaction
